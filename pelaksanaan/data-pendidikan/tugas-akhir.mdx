Menu **Data Tugas Akhir** merupakan submodul yang digunakan untuk menampilkan data **Tugas Akhir mahasiswa** yang terintegrasi dari **sistem Akademik** ke dalam aplikasi **eSPMI**. Data ini digunakan sebagai salah satu data pendukung dalam proses **Sistem Penjaminan Mutu Internal (SPMI)**, khususnya untuk monitoring dan evaluasi capaian mutu akademik.

<Note>
  Admin dapat melihat data **Tugas Akhir mahasiswa** yang telah tersinkron secara otomatis dari sistem Akademik tanpa perlu melakukan input data secara manual.
</Note>

### Cara Mengakses Data Tugas Akhir

1. **Login ke aplikasi eSPMI menggunakan akun Anda.**\
   Silakan akses halaman login e-SPMI dan masukkan akun Anda untuk mulai mengelola data penjaminan mutu.
2. **Pilih menu Pelaksanaan \> Lihat Data Pendidikan**\
   Akan menampilkan beberapa submenu yang di dalamnya menampilkan data yang berasal dari sistem Akademik.
3. **Pilih Data Tugas AKhir**\
   Halaman **Data Tugas Akhir** akan menampilkan daftar data **Tugas Akhir mahasiswa** berdasarkan hasil integrasi dari sistem Akademik. Data yang ditampilkan dapat digunakan sebagai bahan monitoring dan evaluasi capaian mutu akademik pada aplikasi eSPMI.
