Menu **Data Karya Ilmiah** merupakan submodul yang digunakan untuk menampilkan data **karya ilmiah** yang terintegrasi dari **SISTER** ke dalam aplikasi **eSPMI**. Data ini digunakan sebagai salah satu data pendukung dalam proses **Sistem Penjaminan Mutu Internal (SPMI)**, khususnya untuk monitoring dan evaluasi capaian mutu pada bidang penelitian dan publikasi ilmiah.

<Note>
  Admin dapat melihat data **karya ilmiah** yang telah tersinkron secara otomatis dari **SISTER** tanpa perlu melakukan input data secara manual pada aplikasi eSPMI.
</Note>

### Cara Mengakses Data Karya Ilmiah

1. **Login ke aplikasi eSPMI menggunakan akun Anda.**\
   Silakan akses halaman login e-SPMI dan masukkan akun Anda untuk mulai mengelola data penjaminan mutu.
2. **Pilih menu Pelaksanaan \> Lihat Data Penelitian**\
   Akan menampilkan beberapa submenu yang di dalamnya menampilkan data yang berasal dari sistem Sister.
3. **Pilih Data Karya Ilmiah**\
   Halaman **Data Karya Ilmiah** akan menampilkan daftar data **karya ilmiah** berdasarkan hasil integrasi dari **SISTER**. Data yang ditampilkan dapat digunakan sebagai bahan monitoring dan evaluasi capaian mutu pada bidang penelitian dan publikasi ilmiah di aplikasi eSPMI.
