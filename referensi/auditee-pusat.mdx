Menu **Auditee Pusat** digunakan untuk mengelola data Fakultas atau bagian di tingkat pusat yang berperan sebagai **auditee** dalam proses audit mutu internal. Data ini digunakan sebagai referensi dalam pelaksanaan audit pada aplikasi eSPMI.

<Note>
  Data **Auditee Pusat** yang telah digunakan pada proses audit tertentu kemungkinan tidak dapat dihapus untuk menjaga keterkaitan data pada sistem.
</Note>

### Cara Mengakses Menu Auditee Pusat

<AccordionGroup>
  <Accordion title="Login ke aplikasi eSPMI menggunakan akun Anda.">
    Anda dapat login menggunakan username dan password Anda.
  </Accordion>

  <Accordion title="Pilih menu Manajemen Referensi > Auditee Pusat">
    Halaman ini akan menampilkan daftar data **Auditee Pusat** yang telah tersedia pada sistem.

    <Frame>
      ![Auditee Pusat Daftar Fakultas](/images/auditee-pusat-daftar-fakultas.png)
    </Frame>
  </Accordion>

  <Accordion title="Tekan tombol Tambah">
    Anda akan diarahkan pada Form tambah auditee pusat, isi data pada form yang tersedia.

    <Frame>
      ![Auditee Pusat Form Tambah Fakultas](/images/auditee-pusat-form-tambah-fakultas.png)
    </Frame>
  </Accordion>

  <Accordion title="Tekan tombol Simpan">
    Periksa kembali data dan file dokumen yang telah diinput. Data yang Anda simpan akan tampil pada tabel data Auditee Pusat.

    <Frame>
      ![Auditee Pusat Simpan Form Tambah](/images/auditee-pusat-simpan-form-tambah.png)
    </Frame>
  </Accordion>
</AccordionGroup>

### Mengubah & Menghapus Data Auditee Pusat

<AccordionGroup>
  <Accordion title="Pilih data auditee pusat yang ingin diubah.">
    Spesifikasikan data yang ini Anda hapus atau ubah, untuk memastikan data lain tidak berubah.

    <Frame>
      ![Auditee Pusat Pilih Data](/images/auditee-pusat-pilih-data.png)
    </Frame>
  </Accordion>

  <Accordion title="Klik Nama Auditee Pusat">
    Akan muncul tombol tombol **Ubah** dan **Hapus.** Untuk mengubah Anda bisa lakukan klik **Ubah**, lakukan perubahan pada form yang tersedia. Kemudian untuk menghapus data bisa lakukan klik **Hapus** dan data yang Anda pilih untuk hapus akan terhapuskan dari daftar Lembaga Akreditasi.

    <Frame>
      ![Auditee Pusat Form Ubah Hapus Data](/images/auditee-pusat-form-ubah-hapus-data.png)
    </Frame>
  </Accordion>

  <Accordion title="Tekan tombol Simpan">
    Klik tombol **Simpan** untuk menyimpan perubahan data. Apabila proses berhasil, maka data akan diperbarui sesuai dengan perubahan terbaru.
  </Accordion>
</AccordionGroup>
